📚 AI-Study-Assistant
An intelligent learning tool that enhances student productivity by converting notes and study materials into quizzes, flashcards, and audio summaries for better understanding and retention.

🚀 Key Features

Automatic Quiz Creation (MCQs & True/False)

Adjustable Difficulty Levels (Easy, Medium, Hard)

Interactive Flashcards for quick revision

Audio Summaries using Text-to-Speech

Supports file uploads (PDF, TXT, DOCX)

Progress Tracking Dashboard to monitor performance

📦 Setup Instructions

git clone https://github.com/Jahnavimandapalli/-AI-Study-Assistant

cd AI-Study-Assistant

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

Create a .env file and add:

GROQ_API_KEY=your_api_key_here

Run the application:

streamlit run app.py
